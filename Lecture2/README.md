# Анализ взаимосвязи факторов друг с другом. Лекция

## Что делала:
Поиск взаимосвязей между различными факторами в данных, чтобы понять, какие переменные влияют друг на друга, и в будущем использовать их для предсказательных моделей

## Что использовала:
- Библиотеки: pandas, numpy, matplotlib.pyplot, seaborn, math, scipy.stats

## Основные операции:
- Расчет корреляционных матриц для количественных данных
- Использование корреляции Пирсона (для нормально распределенных данных) и Спирмена (для данных с распределением, отличным от нормального) для двух количественных признаков
- Применение коэффициента корреляции Крамера для оценки связи между двумя категориальными признаками
- Расчет корреляционного отношения (ETA) для оценки связи между количественным и категориальным признаками
- Визуализация взаимосвязей с помощью диаграмм рассеяния

## Выводы:
Рассчитаны корреляции между различными комбинациями признаков, выявлены факторы, имеющие наибольшую корреляцию друг с другом, визуализация помогла наглядно оценить характер зависимости между переменными, а также был сделан вывод о том, какие факторы могут быть наиболее значимы для прогнозирования интересующих величин
