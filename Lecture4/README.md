# Проверка гипотез. Лекция

## Что делала:
Применение статистических тестов для проверки гипотез о различиях между группами и о наличии взаимосвязей между переменным, для того чтобы перейти от простого описания данных к обоснованным выводам

## Что использовала:
- Библиотеки: pandas, scipy.stats

## Основные операции:
- Применение критерия Стьюдента (ttest_ind) для нормально распределенных данных и Манна-Уитни для ненормальных, с предварительной проверкой на нормальность с помощью критерия Шапиро-Уилка (shapiro)
- Применение критерия Хи-квадрат (chi2_contingency) для проверки связи между двумя категориальными переменными
- Выбор между корреляцией Пирсона (pearsonr) и Спирмена (spearmanr) на основе проверки нормальности распределения сравниваемых признаков

## Выводы:
Показано, как выбор статистического критерия зависит от типа данных и характера их распределения, и как с помощью этих тестов можно получать обоснованные выводы
