## Описательный анализ количественных и категориальных данных. Лекция
## Что делала:
Первичное знакомство с данными интернет-магазина, их очистка, трансформация и визуализация. Основная цель — подготовить данные к анализу и получить первые описательные статистики

## Что использовала:
-Библиотеки: pandas, numpy, matplotlib.pyplot, seaborn

## Основные операции:
-dropna() для удаления пропусков
-describe() для получения статистик по категориальным и количественным данным.
-groupby() для сравнения показателей в разных группах
-визуализация данных с помощью matplotlib и seaborn

### Выводы:
Данные были успешно загружены и очищены от пропусков. Приведены к единому формату, удобному для анализа. Проведен разведочный анализ (EDA), получены описательные статистики для всех столбцов, что позволило понять распределение данных. Выявлены различия в поведении и выручке между группами покупателей с разной активностью.  Созданы новые категориальные переменные на основе существующих для более детального анализа. Построены графики, наглядно демонстрирующие распределение ключевых показателей
