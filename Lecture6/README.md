# Регрессионный анализ. Выявления значимых факторов. Лекция

## Что делала:
Построение предсказательных моделей, регрессионный анализ. Используя подготовленные ранее данные, обучила модели линейной регрессии для прогнозирования целевых переменных

## Что использовала:
- Библиотеки: pandas, numpy, sklearn.model_selection, sklearn.linear_model, sklearn.preprocessing, sklearn.metrics

## Основные операции:
- Разделение данных на обучающую и тестовую выборки (train_test_split)
- Применение OneHotEncoder для преобразования категорий в числовой формат
- Обучение модели линейной регрессии (LinearRegression)
- Посчитала коэффициент детерминации R^2 и среднюю абсолютную ошибку (MAE), а также среднюю абсолютную процентную ошибку (MAPE)

## Выводы:
Продемонстрирована важность предварительной обработки данных (EDA) для качества модели, получены метрики качества (R^2, MAE, MAPE), позволяющие объективно оценить точность обученных моделей линейной регрессии
