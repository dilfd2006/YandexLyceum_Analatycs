# Как качество исходных данных влияет на аналитические выводы. Лекция

## Что делала:
Решение более сложных задач по предобработке данных (EDA)

## Что использовала:
- Библиотеки: pandas, numpy, scipy.stats

## Основные операции:
- Замена выбросов определенных методом межквартильного размаха на медианное значение по группе
- Заполнение пропусков:
  - модой внутри группы
  - Предыдущим значением (ffill)
  - Методом интерполяции (interpolate)
  - Групповой медианой
- Замена ошибочных названий в категориях
- Создание новых категориальных признаков путем разбиения непрерывных переменных на интервалы

## Выводы:
Отработаны различные стратегии обработки пропусков, применен алгоритм выявления и замены выбросов, созданы новые информативные признаки, которые могут улучшить качество будущих моделей, исправлены ошибки в данных, что повысило их качество и достоверность
