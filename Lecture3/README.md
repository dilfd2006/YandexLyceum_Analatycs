# Генеральная и выборочная выборки. Как правильно сделать вывод. Лекция

## Что делала:
Определение необходимого объема выборки для проведения исследований, сколько данных нужно собрать, чтобы результаты были статистически значимыми

## Что использовала:
-Библиотеки: pandas, numpy, math

## Основные операции:
-Расчет предельной ошибки выборки (H) для доли признака
-Расчет предельной ошибки выборки для среднего значения
-Определение необходимого объема выборки (n) для достижения предельной ошибки при известном доверительном уровне (Z)
-Использование формул для расчета объема выборки

## Выводы:
Освоены формулы и методы для расчета объема выборки, продемонстрировано, как наличие выбросов в данных может влиять на оценку дисперсии и, как следствие, на требуемый размер выборки, а также получены конкретные числовые значения объема выборки для различных сценариев
